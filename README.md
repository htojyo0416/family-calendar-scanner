# 手書きカレンダー予定化プロトタイプ

スマホで手書きカレンダーを撮影し、長女・長男それぞれの予定として確認編集し、カレンダーアプリに取り込める `family-schedule.ics` を書き出すための試作品です。

## 使い方

1. `index.html` をブラウザで開く
2. 長女または長男を選ぶ
3. カレンダー写真を撮影または選択する
4. 読み取り欄に予定テキストを入れる
5. `予定に変換` を押して内容を確認する
6. 必要なら予定名、日付、時間、メモを直す
7. `ICSを書き出す` でスマホのカレンダーに取り込む

## GitHub Pagesでスマホから使う

1. GitHubで新しいリポジトリを作る
2. このフォルダ内のファイルを全部アップロードする
3. リポジトリの `Settings` → `Pages` を開く
4. `Build and deployment` の `Source` を `Deploy from a branch` にする
5. `Branch` を `main`、フォルダを `/root` にして保存する
6. 表示されたURLをスマホで開く
7. iPhoneなら共有メニューから `ホーム画面に追加`、Androidならブラウザメニューからインストールする

公開後のURLは、通常 `https://ユーザー名.github.io/リポジトリ名/` になります。

## 今できること

- スマホカメラから画像を取り込む
- 長女・長男のフォームを切り替える
- `5/12 遠足 9:00-14:00` のような行を予定に変換する
- 手入力で予定を追加・修正・削除する
- iPhone/Googleカレンダーなどに読み込めるICSファイルを書き出す
- GitHub Pagesで公開してスマホのホーム画面に追加する
- 一度開いたあと、基本ファイルをオフラインでも表示する

## 次に足すとよいこと

- 画像をOpenAI Visionなどへ送り、手書き文字を自動で予定候補にする
- 長女・長男それぞれの紙面レイアウトに合わせた読み取りルールを持つ
- GoogleカレンダーやAppleカレンダーへ直接登録する
- 読み取りに自信がない予定を「要確認」として強調する
- 学校行事、習い事、持ち物、締切などの分類を自動で付ける
